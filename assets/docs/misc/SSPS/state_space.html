State Space Problem Solving Modeling: Missionaries and Cannibals

Missionaries & Cannibals

1. PROBLEM STATEMENT

Three missionaries and three cannibals, along with one boat that fits at most two people (and requires at least one person for operation), are on the left bank of a river. The most salient thing about missionaries and cannibals in "cohabitation" is that if ever the cannibals in any one spot (left bank, right bank, on the boat) outnumber the missionaries, the outnumbered missionaries will be consumed - eaten! The goal of this problem is to get all six individuals safely across the river from the left bank to the right bank.


2. THE OBJECTS OF THE WORLD

Analyzing the problem statement yields three types properties per human object in the microworld. These three things are its classification of either missionary or cannibal, it's location, and whether or not it's alive. Additionally, it is necessary to keep track of the boat. This object needs to have location and passenger fields.


3. REPRESENTATION OF A STATE

States in the Missionaries and Cannibals world have 4 properties. An ordered pair to represent the left side of the bank, followed by another order pair to represent the left side of the bank. The first number in the order pair indicates the amount of missionaries on the given of the bank, whereas the second number does the same just for cannibals. You don't need to track whether or not a missionary is alive, because that state would not lead to a solution. The third element of a state set entry is the side of the bank the boat is in, which can either be Left or Right. Finally, the last property of a set entry is an ordered pair with two fields, which is the amount of missionaries on the boat followed by the amount of cannibals on the boat. For example, the goal state would be represented as: { (0, 0), (3, 3), Right, (0, 0) }.


4. THE STATE SPACE DESCRIPTION

1. Initial State -> { (3,3), (0,0), Left, (0,0) }
2. Operators:
  1. Missionary Board Boat - missionary gets on boat
  2. Cannibal Board Boat - cannibal gets on boat
  3. Missionary Off Boat - missionary gets off boat
  4. Cannibal Off Boat - cannibal ges off boat
  5. Switch Sides - state[2] == Left ? Right : Left
3. Goal State -> { (0, 0), (3, 3), Right, (0, 0) }
